
## 1. Words and Expressions

+ Adjustment : 平差
+ Observation Errors ：观测误差
+ Normal Distribution ：正态分布
+ Precision ：精度
+ Accuracy ：准确度
+ Measurement Uncertainty : 测量不确定度

+ Propagation : 传播
+ Variance ：方差
+ Covariance : 协方差
+ Weight : 权

+ Random Errors : 随机误差
+ Systematic Erros : 系统误差

+ Adjustment with Conditions : 条件平差
+ Adjustment of Indirect Observations : 间接平差
